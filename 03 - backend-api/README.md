# Backend API Integration (Node.js)

A full-stack implementation connecting frontend UI with backend APIs to store and retrieve data in a structured format.

---

## 🚀 Features

- API-based data handling  
- Form submission to backend  
- Data storage in structured format  
- Dynamic UI updates from backend responses
